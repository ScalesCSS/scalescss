Media Object Pattern for Scales

This page includes the default media object pattern styles.

Media Object usage

The media object can be used in many situations for positioning any type of image or other media next to some other content. Read Nicole Sullivan's post on the topic http://stubbornella.org/content/2010/06/25/the-media-object-saves-hundreds-of-lines-of-code.
